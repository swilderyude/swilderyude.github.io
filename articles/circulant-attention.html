Transformer模型近年来在计算机视觉领域的应用已取得显著进展。其自注意力机制能有效地捕获图像中的全局依赖关系。然而，自注意力计算的二次复杂度（O(N²)）在高分辨率图像处理时，带来了巨大的计算负担。为此，许多方法通过手工设计注意力模式，如局部注意力或稀疏注意力，以提升计算效率。然而，这些方法通常会牺牲全局表达能力，限制模型的性能。 为了解决这一问题，本文提出了一种名为循环注意力（Circulant Attention）的新机制，通过离散傅里叶变换（DFT）和快速傅里叶变换（FFT）实现高效的自注意力计算。该方法利用BCCB矩阵的性质，实现了O(N log N)的计算复杂度，显著降低了计算开销，同时保持了模型的表达能力。

标准自注意力中的矩阵乘法 在自注意力机制中，假设我们有三个矩阵：

  • ( Q ) (查询矩阵，Query)

  • ( K ) (键矩阵，Key)

  • ( V ) (值矩阵，Value) 它们的维度分别是：

  • ( Q ) 的维度是 ( N × d )，其中 ( N ) 是输入的 token 数量，( d ) 是每个 token 的特征维度。

  • ( K ) 的维度也是 ( N × d )，同样，( N ) 是 token 数量，( d ) 是每个 token 的特征维度。

  • 结果矩阵的维度是 ( N × N )，每个元素表示第 ( i ) 个 token 与第 ( j ) 个 token 的相似度（通过计算它们的点积实现）。这就是所谓的注意力矩阵。 ( Q ) 和 ( K ) 的作用是计算出每个 token 与其他 token 的关系（即注意力分数），而 ( V ) 则提供了每个 token 在最终表示中应该如何被加权。

自注意力的整体过程：

  1. 计算相似度（注意力分数）：通过计算查询矩阵 ( Q ) 和键矩阵 ( K ) 的乘积 ( Q K^T )，得到一个 注意力矩阵。这个矩阵表示了每个 token 与其他 token 的相关性（即相似度）。

  2. 归一化分数：通过 Softmax 对得到的相似度进行归一化，使得注意力权重变成一个概率分布，这样每个 token 对其他 token 的“关注”程度就有了具体的权重。

  3. 加权求和：然后，这些权重会与 值矩阵 ( V ) 进行加权求和。具体来说，经过 Softmax 后的注意力分数矩阵与值矩阵 ( V ) 相乘，最终得到每个 token 的加权表示。这个加权的结果将作为该 token 的最终输出。

因此，( V ) 其实是在计算出每个 token 与其他 token 之间的关系后，用来合并信息的关键部分。最终的加权求和结果就是每个 token 的新的表示，它包含了全局信息，并且是通过关注（加权）其他 token 来得到的。

傅里叶变换（Fourier Transform）的基本概念 傅里叶变换的核心思想是：任何一个周期性的信号都可以用不同频率的正弦波和余弦波加权叠加表示。 在计算机视觉中，数据往往是时序数据或空间数据（如图像）。傅里叶变换能够将这些数据从时域或空间域（原始数据的表现形式）转化为频域（包含不同频率成分的表示）。这种变换能够帮助我们在频域中处理信号，特别是当数据包含周期性变化时。 时域 vs 频域

  • 时域（Time Domain）：这是原始的信号表示（例如图像像素、声音波形等）。

  • 频域（Frequency Domain）：通过傅里叶变换，将时域信号分解成不同频率的正弦波和余弦波的组合。频域中的数据可以更清晰地揭示信号的周期性和频率成分。 在标准自注意力中，计算的是 矩阵乘法，也就是对每个 token 与其他 token 之间的相似度进行加权计算。通过 DFT，将时域的矩阵乘法转化为频域的卷积计算，而频域中的卷积操作实际上变成了元素级的乘法。 计算复杂度为什么变低到 (O(N N))？ 要理解为什么使用 DFT 和 FFT（快速傅里叶变换）可以将计算复杂度降低到 (O(N N))，我们需要看看 FFT（快速傅里叶变换）如何优化离散傅里叶变换（DFT）的计算。 DFT 的时间复杂度： 离散傅里叶变换（DFT）是将时域信号转换为频域信号的基本方法，通常需要 (O(N^2)) 的计算复杂度。如果我们使用直接计算的方式，DFT 的计算复杂度是 (O(N^2))，因为每个输出点都需要计算 (N) 个输入点的加权和。 FFT 的优化： 快速傅里叶变换（FFT）是 DFT 的一种高效实现，能够将计算复杂度降低到 (O(N N))。FFT 通过递归分解和合并计算来减少计算量，它基于分治法，将一个大的 DFT 问题分解成多个较小的 DFT 问题来解决。

  • 分治法：FFT 使用分治法将一个长度为 (N) 的序列分解为多个更小的序列，从而减少了直接计算的工作量。FFT 在每个递归步骤中处理的数据量逐步减小，最终将整个问题的复杂度降低到 (O(N N))。

2 相关工作

视觉 Transformer（Vision Transformer） 近年来，Transformer 架构和注意力机制在计算机视觉领域取得了显著进展（Vaswani et al. 2017）。然而，自注意力的二次计算复杂度在处理全局特征图时会导致难以承受的开销。为了解决这一问题，已有多种方法通过引入人工设计的模式（如局部性或稀疏性）来降低计算负担。 局部方法（如 Swin Transformer（Liu et al. 2021））将注意力限制在不重叠窗口内，从而降低计算成本。CSwin（Dong et al. 2022）通过十字形窗口扩展了这一思想，以获取更丰富的上下文。Neighborhood Attention Transformer（NAT）（Hassani et al. 2023）进一步通过将每个查询的注意力限制在局部邻域来模拟卷积。 除这些设计外，另一条研究路线采用稀疏注意力范式。PVT（Wang et al. 2021）通过对键和值下采样来降低计算复杂度；DAT（Xia et al. 2022）提出了输入依赖的稀疏注意力模式；BiFormer（Zhu et al. 2023）采用双层路由注意力为每个查询动态确定感兴趣区域。 尽管这些方法具有效率优势，但这些人工设计的注意力模式不可避免地会削弱全局自注意力的表达能力。本文利用自注意力的一种内在高效结构，在效率与表达能力之间实现更优平衡。

基于 DFT 的高效架构（Efficient architecture with DFT） 离散傅里叶变换（DFT）长期以来一直是数字图像处理中的重要工具（Pitas 2000）。近期工作将 DFT 与快速傅里叶变换（FFT）用于设计高效网络组件（Li et al. 2021; Rao et al. 2021; Guibas et al. 2021; Huang et al. 2023; Kong et al. 2023）。 GFNet（Rao et al. 2021）利用 DFT 的卷积定理构建全局逐通道卷积模块，并通过 FFT 实现 O(Nlog N) 复杂度。FNO（Li et al. 2021）进一步将该思路推广到稠密全局卷积。AFNO（Guibas et al. 2021）和 AFFNet（Huang et al. 2023）通过输入依赖的频域滤波器实现了等价的动态逐通道卷积。 本文使用二维 DFT 与 FFT 算法来高效实现所提出的循环注意力机制。

3 预备知识

3.1 注意力公式

1. 图像 token 序列 x

x ∈ ℝN × C 输入是一组图像 token。 什么是 token？ 在视觉 Transformer 中，一张图像通常会被切成很多小块，或者经过特征提取后变成很多位置上的特征。 每一个位置的特征就可以看成一个 token。 比如一张特征图大小是： H × W 那么总共有：

N = H × W 个位置，也就是 N 个 token。 每个 token 有 C 个通道特征，所以整个输入写成：

x ∈ ℝN × C

含义是：

  • N：token 数量

  • C：每个 token 的特征维度

2. Q, K, V 是什么？

输入 x 会通过三个投影矩阵：

WQ, WK, WV

得到： Q, K, V ∈ ℝN × d

这就是 Transformer 里的 Query、Key、Value。

符号 | 中文 | 含义 |

|—|—|—|

Q | Query，查询 | 当前 token 用它去“查找”其他 token |
K | Key，键 | 每个 token 提供自己的“被匹配信息” |
V | Value，值 | 最终被加权汇总的信息 |

可以粗略理解为：

  • Q：我想找什么信息

  • K：我拥有什么特征，能不能被你匹配到

  • V：如果你关注我，我实际贡献什么内容 投影矩阵 WQ/K/V

3. 注意力矩阵 A 是什么？

QK⊤ ∈ ℝN × N

这个矩阵的每一个元素表示：

某个 token 的 Query 和另一个 token 的 Key 的相似度。

也就是说，A 里面存的是：

“第 i 个 token 应该多关注第 j 个 token”。

4. 为什么除以 $\sqrt d$？

原文公式：

$$ A=QK^\top/\sqrt d $$

这里的 $\sqrt d$ 是标准自注意力公式中的缩放项。 它把 QK⊤ 的结果进行缩放。

所以 A 是缩放后的原始注意力矩阵。 也就是说，原本 A 只是相似度分数，经过 Softmax 后，每个 token 对其他 token 的关注程度变成权重。 然后再用这些权重去加权求和 V。

5. 输出 O 是什么？

公式：

O = σ(A)V

3.2 循环矩阵

Ci, j = cj − i (mod N) 意思是： 矩阵中第 i, j 个位置的值，可以从第一行 c 中按循环索引找到。 循环矩阵乘向量

$$ (Cx)_i=\sum_{j=0}^{N-1}C_{i,j}x_j $$ 两者相乘得到一个新的 N 维向量：

y = Cx

其中第 i 个元素是：

(Cx)i 第 i 行的每个元素，分别乘以 x 中对应元素，然后全部加起来。 为什么令 k = (j − i) mod N？ 这是一个变量替换。 目的只是把公式写成更像“循环互相关”的形式。 $$ (Cx)_i=\sum_{k=0}^{N-1}c_k\cdot x_{(i+k)\ \mathrm{mod}\ N} $$

对于每个输出位置 i，它做的是：

  • 取 c0, c1, …, cN − 1

  • 分别乘上 xi, xi + 1, …

  • 如果超过末尾，就循环回到开头

  • 最后求和

这就是一种循环方式的匹配计算。 这正是 c 与 x 的一维循环互相关定义。 互相关 互相关可以理解为： 用一个序列去和另一个序列的不同位置进行匹配，然后得到每个位置的匹配结果。 这里是“循环互相关”，所以序列首尾相接。 公式：

$$ (Cx)_i=\sum_{k=0}^{N-1}c_k\cdot x_{(i+k)\ \mathrm{mod}\ N} $$

表示 c 在 x 上以循环方式进行匹配。 为什么说对应 depth-wise 卷积？ 在深度学习中，这对应： “循环填充的一维 depth-wise 卷积”。 这里包含两个名词： depth-wise 卷积 depth-wise 卷积一般指每个通道单独做卷积，不在通道之间混合。 循环填充 循环填充是指序列或特征图边界处用另一端的内容来补齐。 为什么要用傅里叶变换？ 因此 y = Cx 可以由互相关定理计算。 互相关定理 (Cross-Correlation Theorem)：互相关结果的傅里叶变换等价于第一个信号傅里叶变换的共轭和第二个信号的傅里叶变换的元素乘积。 公式： $$ F_{1D}(Cx)=\overline{F_{1D}(c)}\odot F_{1D}(x) $$

然后： $$ Cx=F^{-1}_{1D}\big(\overline{F_{1D}(c)}\odot F_{1D}(x)\big) $$

  1. F1D 是什么？ 表示一维离散傅里叶变换，也就是 1D DFT。

  2. F1D − 1 是什么？

F1D − 1

表示一维离散傅里叶逆变换。 它把频域表示再变回原来的序列表示。

  1. 什么是 DFT？ DFT 是 Discrete Fourier Transform，中文叫离散傅里叶变换。 在原文中，它作为一种计算工具使用。 它可以把循环互相关这种操作转化为频域中的逐元素乘法。

  2. 什么是 $\overline{F_{1D}(c)}$？ 上横线表示复共轭。

$$ \overline{F_{1D}(c)} $$

意思是对 c 做一维 DFT 后，再取复共轭。

  1. 什么是复共轭？ 傅里叶变换后的结果通常是复数。 复数形如： a + bi

它的共轭是： a − bi

也就是虚部符号取反。 论文这里用复共轭是因为互相关定理中需要它。

  1. ⊙ 是什么？ ⊙ 表示逐元素乘法，也叫 Hadamard 乘积。 如果有两个向量：

[a1, a2, a3]

和：

[b1, b2, b3]

逐元素乘法就是：

[a1b1, a2b2, a3b3]

  1. 为什么这样更高效？ 利用快速傅里叶变换算法，可以把 y = Cx 的计算复杂度做到： O(Nlog N) 这里 FFT 是 Fast Fourier Transform，快速傅里叶变换。 它是高效计算 DFT 的算法。

3.3 BCCB 矩阵

3.2 讲的是一维循环矩阵。 3.3 把它推广到二维。 因为图像是二维的，有高度 H 和宽度 W。

BCCB 的全称 带循环块的分块循环矩阵

  1. Block Circulant：块之间是循环的

  2. with Circulant Blocks：每个块内部也是循环矩阵

为什么需要 BCCB？ 前面的一维循环矩阵适合处理一维序列。 但图像特征图是二维的：

H × W

所以需要一种二维版本的循环结构。 BCCB 就是循环矩阵在二维情形下的推广。 BCCB 和二维循环互相关的关系 类似循环矩阵，BCCB 矩阵 B 也可由其第一行完全确定。 b = [c0, c1, …, cHW − 1] 然后把 b, x reshape 成二维：

b̂, x̂ ∈ ℝH × W

  1. reshape 是什么意思？ reshape 就是改变数据形状，但不改变数据本身。 比如一个长度为 H × W 的一维向量，可以重新排成一个 H × W 的二维矩阵。 原文中：

  • b：一维序列

  • x：一维序列 经过 reshape 后：

  • b̂：二维形式

  • x̂：二维形式

  1. Bx 等价于什么？ 也就是说，BCCB 矩阵乘向量，本质上就是二维版本的循环互相关。

  2. 对应深度学习中的什么？ 这对应深度学习中的： “循环填充二维 depth-wise 卷积”。 意思是：

  • 是二维操作，因为处理的是 H × W 特征图

  • 是循环填充，因为边界位置会循环回绕

  • 是 depth-wise 卷积，因为是逐通道形式的卷积操作

4 方法

4.1 视觉 Transformer 中的高效模式

自注意力机制以二次复杂度 O(N2) 计算每个 query 与 key 之间的两两相似性。 然而，我们发现一个有趣现象：在实践中，视觉 Transformer 倾向于学习到更加结构化且高效的注意力模式。具体而言，在图2中，我们可视化了从 DeiT（Touvron et al. 2021）模型中提取的若干注意力矩阵。这些矩阵与带循环块的分块循环矩阵（BCCB）非常相似。正如第3节所述，与 BCCB 矩阵相乘等价于二维全局卷积操作，可通过快速傅里叶变换算法在 O(Nlog N) 时间内完成。这表明：尽管自注意力形式上具有二次复杂度，但它在隐式上继承了一种高效结构。

为更好理解该现象，我们在图3中给出了注意力矩阵中相邻 query token（红点标记）的注意力分布。一个关键观察是：注意力分布随 query 位置移动呈现近似平移不变性，这正是由 BCCB 矩阵实现的二维全局卷积行为。这进一步验证了视觉 Transformer 学到的注意力与 BCCB 矩阵高度相似。

受上述发现启发，一个关键研究问题出现： 我们能否显式地对注意力矩阵施加 BCCB 结构？

在下一节中，我们通过 Circulant Attention（循环注意力） 回答这个问题。该新型注意力机制采用 BCCB 注意力矩阵。通过施加这种结构先验，我们希望将 BCCB 矩阵的有益性质——即 O(Nlog N) 复杂度与内在平移不变性——显式编码进注意力机制，从而同时获得效率与表达能力。

4.2 循环注意力（Circulant Attention）

循环注意力用其最近的 BCCB 矩阵 Ã 来近似原始注意力 A ∈ ℝN × N。形式化地： $$ \tilde A=\arg\min_{B\in\mathcal{B}}\|A-B\|, \tag{9} $$ 其中 ℬ 表示 N × N 的 BCCB 矩阵子空间，∥ ⋅ ∥ 表示 Frobenius 范数。因此，Ã 是 A 在 BCCB 矩阵子空间中的正交投影。 N × N 的 BCCB 矩阵可由其第一行完全决定。记 Bk 为第一行在第 k 个位置为1（one-hot）的 BCCB 矩阵。易知 {B0, ⋯, BN − 1} 构成 BCCB 矩阵子空间的一组基。并且有： $$ \langle B_k,B_k\rangle=N,\quad \langle B_k,B_j\rangle=0,\ k\neq j, \tag{10} $$ 其中 ⟨⋅, ⋅ ⟩ 是 Frobenius 内积（即两个矩阵逐元素乘积之和）。式(10)表明 {B0, ⋯, BN − 1} 是一组正交基。因此，Ã 在 BCCB 子空间中的正交投影可写为： $$ \tilde A =\sum_{k=0}^{N-1}\frac{\langle A,B_k\rangle}{\langle B_k,B_k\rangle}B_k =\frac1N\sum_{k=0}^{N-1}\langle A,B_k\rangle B_k =\frac1N\sum_{k=0}^{N-1}\left\langle\frac{QK^\top}{\sqrt d},B_k\right\rangle B_k. \tag{11} $$ 得到 BCCB 注意力矩阵 Ã 后，我们像标准自注意力一样计算输出： O = σ(Ã)V

高效计算算法。 若直接按投影公式计算，循环注意力仍需 O(N2) 复杂度。这里证明：借助二维 DFT，可在 O(Nlog N) 时间内等价计算。 Ã 是 BCCB 矩阵，其整体结构由第一行决定。记该行向量为 a，则 $$ a=\frac{1}{N\sqrt d}\,[\langle QK^\top,B_0\rangle,\cdots,\langle QK^\top,B_{N-1}\rangle]. \tag{12} $$ 依据 BCCB 的性质，Bk 对应二维空间中的一个位移。定义 $$ \Delta h=\lfloor k/W\rfloor,\quad \Delta w=k\bmod W. \tag{13} $$ 则有 $$ a_k =\frac{1}{N\sqrt d}\langle QK^\top,B_k\rangle =\frac{1}{N\sqrt d}\sum_{h=0}^{H-1}\sum_{w=0}^{W-1} \langle \hat Q_{h,w},\hat K_{h+\Delta h,w+\Delta w}\rangle, \tag{14} $$ 其中 Q̂, K̂ ∈ ℝH × W × d 是 Q, K ∈ ℝN × d 的二维重排版本，并采用循环填充（处理边界）。可以看到，ak 是 Q̂ 与 K̂ 的循环互相关结果，这正是深度学习中的卷积概念。因此，a 可通过二维 DFT 高效计算： $$ a=\frac{1}{N\sqrt d}\Big[F^{-1}_{2D}\big(\overline{F_{2D}(Q)}\odot F_{2D}(K)\big)\Big]\cdot \mathbf{1}_{d\times1} =\frac{1}{N\sqrt d}(Q\circledast K)\cdot \mathbf{1}_{d\times1}, \tag{15} $$ 其中 ⊛ 是定义的 DFT 乘法，且 Q ⊛ K ∈ ℝN × d。1d × 1 ∈ ℝd × 1 是全1向量，对应式(14)中 ⟨Q̂h, w, K̂h + Δh, w + Δw⟩ 在维度 d 上的隐式求和。 由于 Ã 是 BCCB 矩阵，且 σ 是按行的 Softmax，σ(Ã) 也仍是 BCCB 矩阵，且 σ(a) 是其第一行。 可高效写为： $$ O=F^{-1}_{2D}\big(\overline{F_{2D}(\sigma(a))}\odot F_{2D}(V)\big) =\sigma(a)\circledast V. \tag{16} $$ 在图1中，我们给出了所提循环注意力范式示意图。可见该设计在结构上与标准自注意力相似，关键差别在于使用 BCCB 注意力矩阵以及基于 DFT 的高效乘法。

复杂度分析。 利用高效算法，循环注意力总复杂度为： Ω(CA) = 2N(log2N)d + 2Nd + Nlog2N + N(log2N)(d + 1) + 2Nd + N(log2N)d $$ = N(\log_2N)(4d+2)+4Nd, \tag{17} $$ 显著优于标准自注意力： $$ \Omega(\mathrm{SA})=N^2d+N^2d=2N^2d. \tag{18} $$ Token 重加权模块。 在标准自注意力中，对原始注意力 A ∈ ℝN × N 做按行 Softmax，保证每行和为1。但这不会约束 σ(A) 的列和，因此某些 key 可在跨 query 的聚合中获得更高总注意分数。于是，不同 query 可共同强调某些 key/value，有助于区分更有信息量的 token。 但在我们的循环注意力中，σ(Ã) 的分块循环结构会同时约束行和与列和都为1，从而限制其突出显著 token 的能力。为此，引入简单有效的 token 重加权。具体有两种形式： 预重加权： $$ O=\mathrm{CirAttn}(Q,K,V\odot T), \tag{19} $$ 后重加权： $$ O=\mathrm{CirAttn}(Q,K,V)\odot T, \tag{20} $$ 其中 T = SiLU(xWT) ∈ ℝN × d 是输入相关的 token 重加权因子，CirAttn 表示循环注意力算子。

4.3 实现细节

循环注意力是一个可插拔模块，可用于多种视觉 Transformer。作为展示，我们在三类代表性结构上实现：DeiT（Touvron et al. 2021）、PVT（Wang et al. 2021）和 Swin Transformer（Liu et al. 2021）。三者分别代表全局自注意力、稀疏注意力和局部注意力。我们将这些模型中的原注意力模块替换为循环注意力。 由于本方法具有 Nlog N 复杂度，可在早期阶段直接处理高分辨率特征图而不过多增加计算开销。因此，对 Swin、PVT 这类层级模型，我们主要在前两个 stage 替换注意力。除基线改进外，我们还构建了专用模型家族 CAT（Circulant Attention Transformer），用于与先进视觉 Transformer 对比。详细结构见附录。

论文说：普通自注意力要算一个 N × N 的注意力矩阵（每个token和每个token都比一下），所以复杂度是 O(N2)，很贵。 作者观察到：模型学出来的注意力矩阵，很多时候“长得像”一种特殊矩阵 BCCB。这种矩阵可以用FFT快速算，复杂度可到 O(Nlog N)。

推理公式

Step 1：先算普通原始注意力矩阵 $$ A=\frac{QK^\top}{\sqrt d} $$ 这是和标准注意力一样的起点。 Step 2：把 A 投影到 BCCB 子空间，得到 Ã Ã = arg minB ∈ ℬ∥A − B∥

  • 原始 A 不一定是 BCCB，直接 FFT 快算不了；

  • 但 Ã 是 BCCB，又尽量贴近 A；

  • 这样既保留原注意力信息，又能用 O(Nlog N) 计算。 Step 3：用BCCB基向量写出这个投影（式11）

论文构造一组基 B0, …, BN − 1：每个 Bk 是一个BCCB矩阵，其第一行是第k位为1的one-hot。 这组基是正交的（式10），所以投影可写成 $$ \tilde A=\frac1N\sum_{k=0}^{N-1}\langle A,B_k\rangle B_k =\frac1N\sum_{k=0}^{N-1}\left\langle\frac{QK^\top}{\sqrt d},B_k\right\rangle B_k. $$ 含义：Ã 由每个基方向上的“投影系数”加权得到。 $$ \tilde A=\frac1N\sum_{k=0}^{N-1}\langle A,B_k\rangle B_k $$

  1. 这个 k 到底是什么？ k 就是第几个基向量（基矩阵）的编号。 类似：

  • 一维坐标系里基是 e1, e2, …

  • 这里的“基”不是向量而是矩阵：B0, B1, …, BN − 1 所以 k = 0, 1, …, N − 1 只是“挨个枚举每个基矩阵”。

  1. “每个 Bk” 是怎么来的？ 论文定义： Bk 是一个 BCCB 矩阵，且它的第一行是 one-hot：第 k 个位置是1，其它是0。 然后根据 BCCB 规则，其余行列由循环结构自动确定。 所以：

  • B0：第一行在第0位是1

  • B1：第一行在第1位是1

  • …

  • BN − 1：第一行在最后一位是1 这 N 个矩阵张成了 BCCB 子空间（可理解成“这类结构的坐标轴”）。

  1. 式子在做什么？ ⟨A, Bk⟩ 是“A 在第 k 个基方向上的分量大小”（投影系数）。 然后 ⟨A, Bk⟩Bk 表示沿第 k 个基方向重建出来的一部分。

最后对所有 k 求和，就是把所有方向拼起来，得到投影 Ã。

  1. 为什么前面有 $\frac1N$？ 因为这组基是“正交但范数不是1”的规范化方式。 在这篇里有 ⟨Bi, Bj⟩ = Nδij 所以做投影时要除以 N（如果是单位正交基就不用除）。

整个式子就是把 A 在这 N 个基矩阵上做正交投影再重构。 Step 4：只需求 Ã 的第一行 a（因为BCCB由第一行唯一决定）

$$ a=\frac{1}{N\sqrt d}\big[\langle QK^\top,B_0\rangle,\dots,\langle QK^\top,B_{N-1}\rangle\big]. $$

第k个元素 ak 可写成二维位移相关（式13、14）：

  • Δh = ⌊k/W⌋, Δw = k mod  W

  • ak 是 Q 和 K 在该位移下的内积求和（循环边界） 这说明 ak 本质是循环互相关结果。 Step 5：把上一步改成DFT高效算（式15） 因为是循环互相关，可用2D DFT： $$ a=\frac{1}{N\sqrt d}\Big[F^{-1}_{2D}\big(\overline{F_{2D}(Q)}\odot F_{2D}(K)\big)\Big]\cdot \mathbf{1}_{d\times1} $$ 论文记为 $$ a=\frac{1}{N\sqrt d}(Q\circledast K)\cdot \mathbf{1}_{d\times1}. $$ 含义：先频域逐元素乘，再逆变换回空间域，并在通道维 d 上求和。 Step 6：计算输出 O（式16） 有了 Ã 后，仍然按注意力形式： O = σ(Ã)V. 由于 Ã 是BCCB，σ(Ã) 仍是BCCB，其第一行是 σ(a)。 因此同样可用DFT快速算： $$ O=F^{-1}_{2D}\big(\overline{F_{2D}(\sigma(a))}\odot F_{2D}(V)\big) =\sigma(a)\circledast V. $$ Step 7：复杂度结论（式17、18）

  • 标准自注意力：Ω(SA) = 2N2d

  • 循环注意力：Ω(CA) = N(log2N)(4d + 2) + 4Nd

所以是 Nlog N 级别，相比 N2 更高效。

  1. 标准自注意力为什么是 2N2d 忽略 softmax、线性层，只看核心两步：

  2. QK⊤： Q ∈ ℝN × d, K ∈ ℝN × d ⇒ K⊤ ∈ ℝd × N 乘法代价  ∼ N ⋅ N ⋅ d = N2d

  3. AV： A ∈ ℝN × N, V ∈ ℝN × d 代价  ∼ N ⋅ N ⋅ d = N2d

要算 O = AV,  A ∈ ℝN × N, V ∈ ℝN × d, 输出 O ∈ ℝN × d。 1）先看一个元素怎么来的 O 的第 i 行第 j 列： $$ O_{ij}=\sum_{k=1}^{N}A_{ik}V_{kj}. $$

这就是：

  • 把 A 的第 i 行（长度 N）

  • 和 V 的第 j 列（长度 N） 做一次点积。 一次点积长度是 N，代价约 N 次乘加。 2）这样的元素有多少个？ O 是 N × d，所以一共有 Nd 个元素。 每个元素要花  ∼ N 代价。 总计： Nd × N = N2d. Step 8：Token Reweighting（补偿模块） 论文指出： 标准注意力行softmax后，列和不受限，某些key可被多query共同强调。 而BCCB结构下会更受限（行列和都为1），突出“显著token”的能力会受影响。 所以加一个重加权因子 T = SiLU(xWT) ∈ ℝN × d 在普通 attention 里：

  • 每个 query 的一行 softmax 后和为 1（行归一）

  • 但“列”不受限，所以同一个 key/token 可以被很多 query 同时高权重引用 => 显著 token 容易被“全局放大”。

在你这个 CirAttn（BCCB 约束）里，权重结构更“均匀/受限”，这种“全局集中火力”能力会弱一点。 所以加一个门控 T，让模型自己学“哪些 token/channel 该放大”。

T = SiLU(xWT) 是什么？

  • 输入 x: [N, d]

  • WT: [d, d]（常见设定）

  • xWT: [N, d]

  • SiLU 激活后还是 [N, d] 所以 T[n, c] 就是一个逐 token、逐通道的缩放系数（像音量旋钮）。

两种重加权

  1. 前重加权（Pre-weight） O = CirAttn(Q, K, V ⊙ T) 代码感知：
展开代码PYTHON
T = silu(x @ W_T)        # [N,d]
V2 = V * T               # [N,d] 逐元素乘
O  = CirAttn(Q, K, V2)   # [N,d]

含义： 先把 V 里重要位置放大，再送进注意力混合。 => 会影响“信息如何被传播和混合”的全过程。

  1. 后重加权（Post-weight） O = CirAttn(Q, K, V) ⊙ T

代码感知：

展开代码PYTHON
T = silu(x @ W_T)         # [N,d]
O0 = CirAttn(Q, K, V)     # [N,d]
O  = O0 * T               # [N,d]

含义： 先做完注意力，再对输出做门控。 => 不改变注意力内部混合，只在最后“调音量”。

5 实验

在本节中，我们进行了大量实验来评估我们方法的有效性，包括 ImageNet 分类（Deng et al. 2009）、COCO 目标检测（Lin et al. 2014）以及 ADE20K 语义分割（Zhou et al. 2019）。同时还提供了可视化与分析。

5.1 ImageNet 分类

ImageNet-1K数据集包含 128 万张训练图像和 5 万张验证图像，共 1000 个类别。为确保公平比较，我们采用与基线模型完全一致的训练设置。具体来说，模型从头训练 300 个 epoch，使用 AdamW（Loshchilov and Hutter 2018）优化器，学习率采用余弦衰减，并在前 20 个 epoch 进行线性 warm-up。初始学习率设为 1 × 10 − 3，权重衰减为 0.05。数据增强与正则化策略包括 RandAugment（Cubuk et al. 2020）、Mixup（Zhang et al. 2018）、CutMix（Yun et al. 2019）和随机擦除（Zhong et al. 2020）。

从表 2 左侧可以看到，在三个代表性模型中将自注意力替换为循环注意力后，均带来了一致的提升。比如，CA-DeiT 优于全局自注意力基线 DeiT（Touvron et al. 2021）。这表明我们的方法不仅具有高效率，也有助于视觉 Transformer 的学习。进一步地，与稀疏注意力 PVT（Wang et al. 2021）和局部注意力 Swin Transformer（Liu et al. 2021）相比，循环注意力也展现出显著优势。我们的 CA-PVT-S 仅用 PVT-L 30% 的参数量和 40% 的 FLOPs，即可达到与 PVT-L 相当的准确率。这些结果表明循环注意力是自注意力的一个有前景的替代方案。

表 2 右侧将我们的 CAT 模型与多种最新视觉 Transformer 及 Mamba 变体进行比较。可以看到，循环注意力优于高度优化的方法，验证了我们方法更强的能力。

  • 参数量 (M): 模型的参数数量，以百万（M）为单位，表示模型的复杂度。

  • FLOPs (G) 浮点运算量: 每次前向推理所需的浮点运算次数，以十亿（G）为单位，反映模型的计算需求。

  • Top-1 准确率 (%): 在分类任务中，模型预测正确的概率，表示模型的性能。

5.2 目标检测

COCO（Lin et al. 2014）目标检测与实例分割数据集包含 11.8 万张训练图像和 5000 张验证图像。我们遵循基线模型的训练与测试策略。结果见表 3。

循环注意力以 O(Nlog N) 复杂度提供有效全局建模，这使其非常适用于高分辨率图像建模场景。值得注意的是，CA-PVT-S 在显著更低 FLOPs 下比更大的 PVT-L 高出 1.3 box AP。这一增益比分类任务中更明显（分类任务中 CA-PVT-S 与 PVT-L 准确率相同）。这些结果突出了我们方法在高分辨率场景下的优越性。

5.3 语义分割

ADE20K（Zhou et al. 2019）是一个广泛使用的语义分割基准，包含 2 万张训练图像、2000 张验证图像和 150 个语义类别。我们采用与基线模型相同的设置。与目标检测任务类似，表 4 结果表明循环注意力能够稳定提升 PVT 与 Swin 模型的性能。在计算代价和参数量相近或更低的情况下，我们观察到最高 3.7% 的 mIoU 提升。

5.4 分析与可视化

效率分析。 受益于 O(Nlog N) 复杂度，我们的循环注意力相较自注意力展现出显著效率优势。如图 4(a) 所示，在 15362 图像分辨率下，CA-DeiT-T 的 FLOPs 需求减少 8 倍，显示出其在高分辨率建模任务中的潜力。并且，这些理论节省直接转化为实际收益：图 4(b) 中在 15362 分辨率下实现了 7 倍加速。此外，图 4(c)(d) 显示，在默认 ImageNet 分辨率 2242 下，我们模型在吞吐量与准确率之间取得更优折中，在性能提升的同时，推理速度最高可提升 1.5 倍。

可视化。 如第 3.3 节所述，BCCB 矩阵对应二维全局卷积操作。因此，可以方便地可视化并解释其等效全局卷积核。如图 5 所示，我们的模型可根据输入图像生成多样化的等效卷积核。例如，给定第一张图像，模型生成了鸟形和线状卷积核（前两个）。也存在更多关注空间模式的卷积核，如局部、全局、半平面、条带和十字形等。

5.5 消融研究

我们进行消融实验以评估循环注意力各个设计组件的贡献。实验在 ImageNet-1K 上、基于 CA-DeiT 框架完成。如表 5 所示，我们从 DeiT-S 基线开始，依次加入各项设计。仅引入循环注意力会带来可忽略的 0.1% 性能下降，这表明 BCCB 模式适用于视觉 Transformer，且不会牺牲表达能力。

在此基础上，我们发现该设计受益于较小 head 维度和更多 heads。将 head 维度设为 d = 1 后，准确率提升到 80.2%，超过 DeiT-S 基线。这可归因于循环注意力分数是 N 组 query-key 对求和（见式 (14)），因此其等效 head 维度为 Nd，即使 d = 1 也依然足够大。

另外，我们研究了式 (19) 与式 (20) 中定义的两种 token 重加权方法。两种设计都能进一步提升性能，其中后重加权略优。值得注意的是，token 重加权在 DeiT-S 基线上提升有限。本文默认采用后重加权，在 DeiT-S 上达到 81.0% 准确率。这些结果验证了我们设计的有效性。

Table 3（COCO 检测/分割）表头

Backbone | FLOPs | Sch. | AP^b | AP^b_50 | AP^b_75 | AP^m | AP^m_50 | AP^m_75

  • Backbone：骨干网络 读：拜克本 / 骨干

  • Sch.（Schedule）：训练日程（如 1x/3x） 读：斯凯久 / 训练策略

  • AP^b：检测框平均精度（box AP） 读：A-P box

  • AP^b_50 / AP^b_75：IoU=0.50 / 0.75 时的框 AP 读：A-P box 五十 / 七十五

  • AP^m：实例分割掩码 AP（mask AP） 读：A-P mask

  • AP^m_50 / AP^m_75：IoU=0.50 / 0.75 时的 mask AP

Table 4（ADE20K 分割）表头

Backbone | Method | FLOPs | #Params | mIoU

  • mIoU（mean Intersection over Union）：平均交并比 读：艾米-艾欧优 / 平均交并比

Table 5（消融实验）表头

FLOPs | #Param | Acc. | Diff.

  • #Param：参数量（单数写法，本质同 #Params）

  • Acc.（Accuracy）：准确率 读：艾克尤若西

  • Diff.（Difference）：相对差值/变化量 读：迪夫

6 结论

本文揭示了视觉 Transformer 中一个有趣的模式：其自注意力图经常呈现近似的分块循环且块内循环（BCCB）结构。

这一观察直接启发了我们设计 Circulant Attention（一种新的注意力范式），它显式利用这种内在模式，在保持表达能力的同时优化计算效率。

在分类、目标检测和语义分割任务上的大量实验充分证明了所提出循环注意力的有效性，并将其确立为视觉 Transformer 架构中广泛使用的自注意力的一种高效且有竞争力的替代方案。
