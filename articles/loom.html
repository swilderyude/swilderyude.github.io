Obsidian DataLoom 使用笔记

posted @ 2026-05-28 swilderyude 文件夹: CS自学 / 根目录笔记

整理 DataLoom 表格、字段类型、过滤排序、Source 自动生成行和导出功能。

Obsidian DataLoom 知识管理

一、创建一个 Loom（表格）                                                     

  三种方式任选其一：                                                            

  1. 命令面板（Ctrl/Cmd + P）→ 输入 DataLoom: Create new lom                    

  2. 文件浏览器中右键某个文件夹 → “New lom”

  3. 在某个 Markdown 笔记里光标处插入：命令面板搜 DataLoom: Create new loom and embed it into current file，or仅搜embed，会以嵌入块的形式出现                              

  新建后会得到一个 .loom 文件，本质是 JSON，但用 DataLoom 视图打开就是可视化表格。

  二、表格基本操作

  - 新增行：表格底部点 New row                                                  

  - 新增列：表格最右侧点 New column

  - 改列类型：点列名 → 选 `Type → 切换字段类型                                  

  - 拖动调整列宽：拖列分隔条；拖列名调整列顺序

  - 菜单：每个单元格、列头、行头点击都会弹出对应菜单                            

  三、字段类型（最常用）                                                        

  | 类型 | 用途 |                                                               

  |—|           

  | Text | 普通文本 |

  | Number | 数字 |

  | Tag | 单选标签 |                                                            

  | Multi-tag | 多选标签 |

  | Date | 日期，带日历选择器 |                                                 

  | Checkbox | 勾选框 |

  | File | 链接到 vault 中的文件（[wikilink]]） |                               

  | Embed | 嵌入图片/文件预览 |                                                 

  | Currency | 货币（可选符号） |                                               

  | Last edited time / Creation time | 自动时间戳 |                             

  | Source / Source file | 行来源（配合“从文件夹生成行”功能） |                 

  四、视图过滤、排序、搜索                                                      

  - 筛选：左上角 Filter 按钮 → 添加规则（按字段、运算符、值）                   

  - 排序：点列头 → Sort ascending / descending，可多列叠加

  - 搜索：右上角放大镜，跨所有单元格全文搜索                                    

  - 切换视图：当前主要是 Table 视图；列可以隐藏（列菜单 → Toggle visibility）   

  五、Source（自动从文件夹生成行）⭐ 这是它的杀手锏                             

  打开表格右上角设置 → Sources → Add source：                                   

  - Folder source：选一个文件夹，DataLoom                                       

  会把里面的每个笔记自动作为一行；新增/删除笔记，表格自动同步

  - Frontmatter 映射：笔记 YAML frontmatter 里的字段，可以映射到 DataLoom       

  的列上自动填充                                                                

  这样你可以把整个文件夹的笔记当成一个数据库管理（类似 Notion 的 Database 或    

  Dataview）。    

  六、嵌入到笔记                                                                

  在普通笔记里写：                                                              

  [我的表格.loom]]

  就会把这个 lom 渲染进当前笔记。也可以用 DataLoom: Create new loom and embed it

   into current file 命令一步到位。                                             

  七、导出        

  命令面板搜 DataLoom: Export → 支持导出 Markdown、CSV。                        

  八、常用快捷键（在表格内）                                                    

  - Enter：进入单元格编辑                                                       

  - Esc：退出编辑

  - Tab / Shift+Tab：跳到下一/上一单元格                                        

  - ↑↓←→：移动选中单元格                                                        

  —                                                                           

  入门建议        

  第一次用，建议这么走一遍：

  1. 新建一个 lom，叫 读书清单                                                  

  2. 加几列：书名 (Text)、作者 (Tag)、状态 (Tag: 在读/已读/想读)、评分 

  (Number)、完成日期 (Date)                                                     

  3. 录入几行，玩一下筛选和排序

  4. 再做一个 Source: Folder                                                    

  试——指向你的某个笔记文件夹，看它怎么自动把笔记拉成行                          

  走通这套就基本掌握了，剩下的公式、聚合（Calculation 行：底部点字段下方可以选  

  Sum/Average/Count 等）顺手就会。
