自定义数据集读取代码

from torch.utils.data import Dataset
from PIL import Image
import os

class MyData(Dataset):
    
    def __init__(self, root_dir, label_dir):
        self.root_dir = root_dir
        self.label_dir = label_dir
        self.path = os.path.join(self.root_dir, self.label_dir)
        self.img_path = os.listdir(self.path)
        
    def __getitem__(self, idx):
        img_name = self.img_path[idx]
        img_item_path = os.path.join(self.root_dir, self.label_dir, img_name)
        img = Image.open(img_item_path)
        label = self.label_dir
        return img, label

(注：代码没有引入 Dataset，在真实运行环境中，通常需要在开头加上 from torch.utils.data import Dataset) self 将变量声明为生命周期随对象存续且在类内部全局可见的实例属性，不加则将其限定为仅在当前方法执行期间有效的局部变量。

  1. 导入工具包

  • PIL (Python Imaging Library): 这是Python中专门用来处理图像的工具包。Image 模块可以打开、读取图片。

  • os: 这是专门用来和操作系统（比如Windows的系统路径、文件夹等）打交道的工具包。

  1. 定义一个类
class MyData(Dataset):

类继承。

  1. 初始化方法
    def __init__(self, root_dir, label_dir):
        self.root_dir = root_dir
        self.label_dir = label_dir
        self.path = os.path.join(self.root_dir, self.label_dir)
        self.img_path = os.listdir(self.path)

带有双下划线的 __init__ 是Python的构造方法，它会自动且最先运行。 前后都带双下划线的方法叫 “magic method” 或 “dunder method”（double underscore），由 Python 解释器在特定时机自动调用。__init__、__str__、__len__ 等都属于这一类。

  • self: 代表“自己”。

  • root_dir 和 label_dir: 在启动时需要的两个地址。root_dir 是主文件夹 “dataset”，label_dir 是子文件夹 “cats” (猫)。

  • os.path.join: 路径拼接工具。它会把 root_dir 和 label_dir 拼成一个完整的路径（比如 dataset/cats）。它比直接用加号拼字符串好，因为它知道Windows用 \，Mac用 /，绝不会拼错。

  • os.listdir(self.path): 它的作用是打开上面拼好的路径文件夹，把里面的所有图片文件名列成一个列表。

  1. 获取具体数据的方法
    def __getitem__(self, idx):
        img_name = self.img_path[idx]
        img_item_path = os.path.join(self.root_dir, self.label_dir, img_name)
        img = Image.open(img_item_path)
        label = self.label_dir
        return img, label

__getitem__ 也是一个魔法方法。当机器收到指令“给我第 idx 个数据”时，就会触发这段代码。

  • idx: 意思是 Index（索引、序号）。如果你想要清单里的第0张图，idx 就是 0。

  • img_name = self.img_path[idx]: 从刚才整理好的文件名清单里，查出第 idx 个文件的名字。

  • img_item_path = ...: 再次使用路径拼接，把主文件夹、子文件夹、图片名字拼成一个最终的绝对路径。

  • Image.open(...): 调用上面导入的图片工具包，真正地把这张图片打开，存入变量 img 中。

  • label = self.label_dir: 给这张图片贴上标签。在这里，它直接拿所在文件夹的名字作为这张图的标签。

  • return img, label: 把处理好的图片本身和它的标签一起给出。
