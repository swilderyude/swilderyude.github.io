基本操作

窗口的展示

API namedWindow(名字，属性):创建窗口 imshow:在窗口中显示图片 destroyAllWindows:销毁窗口 resizeWindow:窗口大小 看参数 源码：opencv-master vi modules/highgui/include/opencv2/highgui.hpp 示例：

python title:"test" import cv2 Cv2. namedWindow('new', cv2.WINDOW_AUTOSIZE) # 窗口大小不能改变，WINDOW_NORMAL可以改变窗口大小 # WINDOW_NORMAL参数时 # cv2. resizeWindow('new', 1920, 1080) cv2. imshow('new' , 0) # 0时图片是黑色 key = cv2. waitKey (O) # 0代表一直显示 if (key = 'q'): exit cv2. destroyAllWindows ()
