cd ～/.ssh mkdir 创建新的目录（文件夹） git init 初始化当前目录为git 项目 git status 查看当前项目 git 状态 git add .  把当前路径及子路径下所有内容加入本地git 暂存区 或git add “文件名” git commit -m “初始化项目” / git commit -m “添加说明” 提交到暂存区 git remote add origin  xxx 添加一个远程仓库关联  网址ssh模式不需要密码 /git remote set-url origin xxx更改远程地址 git push -u origin master/main /git push git push -f origin master（强制会覆盖）

git push

添加大文件 先把超过50M的大文件分出来，把其他的按上述步骤传上去。 git lfs install 初始化 LFS（为当前仓库启用） git lfs track “文件名” Git LFS 跟踪 git add .gitattributes git add “文件名” git commit -m “添加大文件” git push origin master

git branch -vv 查看分支信息 git remote -v 查看远程信息 git push –set-upstream origin master 把远程分支作为当前分支推送的关联分支 git config –global –unset http.version 清除git缓存） git上传大文件或添加文件 https://blog.csdn.net/wzk4869/article/details/131661472 git修改文件路径https://m.w3cschool.cn/githubcn/githubcn-dxoy3ato.html git添加文件
